# 한콘연 오늘의 Best Image Top30 - 사용자 가이드

---

## 일반 사용자 가이드

### 투표 탭
- 진행 중인 콘테스트의 이미지를 확인하고 마음에 드는 작품에 투표
- 1인 1표 (브라우저 기준), 투표 후 변경 불가
- 투표 시 "이 작품에 투표하시겠습니까?" 확인 후 진행
- 상단에 투표 마감까지 남은 시간 카운트다운 표시
- 진행 중인 콘테스트가 없으면 빈 화면

### 결과 탭
- 투표 진행 중: "오늘의 Best Image는 잠시 후에 공개 됩니다." 메시지
- 투표 종료 후 관리자 선정 전: "최종 결과는 잠시 후에 발표 됩니다." 메시지
- 관리자 선정 완료: 최종 1위 크게 표시 + 나머지 참여 작품 그리드

### 아카이브 탭
- 캘린더에서 과거 콘테스트 날짜 확인 (파란 점 표시)
- 날짜 클릭 시 Top3 미리보기 + 전체 작품 표시
- "전체 결과 보기" 버튼으로 상세 페이지 이동

## 관리자 가이드

### 접속
- `/admin` 경로로 접속
- 비밀번호 입력 (초기 비밀번호: `000000`)
- 탭/브라우저 종료 시 자동 로그아웃

### 콘테스트 운영 순서

| 단계 | 작업 | 설명 |
|------|------|------|
| 1 | 콘테스트 생성 | 날짜 선택 후 "새 콘테스트 생성" |
| 2 | 이미지 업로드 | 드래그앤드롭 또는 클릭으로 업로드 |
| 3 | 투표 시간 설정 | 시작/종료 시간 설정 후 "시간 저장" |
| 4 | 투표 시작 | 설정 시간에 자동 시작 또는 수동 "투표 시작" 버튼 |
| 5 | 투표 종료 | 수동 "투표 종료" 버튼 |
| 6 | 최종 1위 선택 | 종료 후 나타나는 이미지 클릭 → 확인 → 결과 탭에 반영 |

### 주요 기능

#### 투표 시간 설정
- 시작/종료 시간 지정 가능
- 종료 시간 기본값은 시작 시간 1시간 후

#### 자동 투표 시작
- 설정된 시작 시간이 되면 자동으로 투표 시작 (관리자 페이지가 열려있을 때)

#### 수동 투표 시작
- 시작 시간 전에 "투표 시작" 버튼을 누르면 "N분 남았습니다. 정말 시작하시겠습니까?" 확인 후 진행

#### 최종 1위 선택
- 투표 종료 후 "최종 1위 선택" 섹션이 나타남
- 이미지를 클릭하면 "최종 1위로 선택하시겠습니까?" 확인
- 선택된 1위는 왕관 아이콘으로 표시
- 다른 이미지를 클릭하여 재선택 가능

#### 투표수 확인 (관리자 전용)
- 업로드된 이미지 목록에서 각 이미지의 투표수 확인 가능
- 상단에 총 투표수 표시
- 투표수는 관리자 페이지에서만 볼 수 있으며, 일반 사용자에게는 노출되지 않음

#### 지난 콘테스트 관리
- 캘린더에서 날짜 선택
- 상태 변경 (준비중 / 진행중 / 종료)
- 콘테스트 삭제 (이미지, 투표 데이터 포함 전체 삭제)

#### 비밀번호 변경
- 우측 상단 "비밀번호 변경" 버튼
- 현재 비밀번호 확인 후 새 비밀번호 설정 (4자 이상)
