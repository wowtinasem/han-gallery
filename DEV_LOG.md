# 한콘연 오늘의 Best Image Top30 - 개발 기록

---

## 프로젝트 개요

- **프로젝트명**: 한콘연 오늘의 Best Image Top30
- **기술 스택**: Next.js 16 (App Router), Firebase/Firestore, Cloudinary, Tailwind CSS
- **배포**: Vercel (https://han-gallery.vercel.app)

---

## 구현 기능 목록

### 1. 관리자 캘린더 기반 아카이브 관리

- 리스트 뷰에서 캘린더 UI로 변경
- 월별 네비게이션 (◀ ▶)
- 날짜 선택 시 해당 콘테스트 상세 정보 표시
- 상태 변경 (준비중 / 진행중 / 종료)
- 콘테스트 삭제 (이미지, 투표 데이터 포함 전체 삭제)

### 2. 일반 사용자 캘린더 아카이브

- 캘린더에서 과거 콘테스트 날짜 확인 (파란 점 표시)
- 날짜 클릭 시 Top3 미리보기 + 전체 작품 그리드 표시
- "전체 결과 보기" 버튼으로 상세 페이지 이동
- 읽기 전용 (편집/삭제 불가)

### 3. 모바일 반응형 UI

- **네비게이션**: 모바일에서 로고 숨김, '한국AI콘텐츠연구소' 텍스트만 표시, 토글형 탭
- **히어로 섹션**: 모바일 `text-2xl` / PC `text-5xl` 반응형 폰트
- **투표 이미지**: 모바일 1열, PC 2열 이상 그리드

### 4. 결과 탭 3단계 표시

- **투표 진행 중**: "오늘의 Best Image는 잠시 후에 공개 됩니다." 대형 메시지
- **투표 종료 + 관리자 미선정**: "최종 결과는 잠시 후에 발표 됩니다." 메시지
- **관리자 선정 완료**: 최종 1위 크게 표시 (왕관/금색 테두리) + 나머지 참여 작품 동일 레벨 그리드 (투표수 미표시)

### 5. 관리자 최종 1위 선택 시스템

- Contest 타입에 `winnerId` 필드 추가
- 투표 종료 후 "최종 1위 선택" 섹션 표시
- 이미지 클릭 → "최종 1위로 선택하시겠습니까?" 확인 → 결과 탭에 반영
- 선택된 1위는 왕관 아이콘으로 표시, 재선택 가능

### 6. 자동 투표 시작

- 설정된 시작 시간이 되면 자동으로 투표 시작 (10초 간격 체크, 관리자 페이지 열린 상태)
- 시작 시간 전 수동 "투표 시작" 클릭 시 "N분 남았습니다. 정말 시작하시겠습니까?" 확인 후 진행
- 종료 시간 기본값: 시작 시간 + 1시간

### 7. 관리자 전용 투표수 확인

- 업로드된 이미지별 투표수 표시 (주황색)
- 총 투표수 상단에 표시
- 일반 사용자에게는 투표수 노출되지 않음

### 8. 기타

- 투표 시 "이 작품에 투표하시겠습니까?" 확인 다이얼로그
- 콘테스트 삭제 시 하위 컬렉션(이미지, 투표) 포함 전체 삭제
- 관리자 비밀번호 Firestore 기반 관리 (변경 가능)

## 알려진 제한사항

- **자동 투표 시작**: 관리자 페이지가 열려있을 때만 동작 (클라이언트 사이드 폴링 방식)
- **1인 1표**: 브라우저 핑거프린트 기반으로, 다른 브라우저/기기에서는 중복 투표 가능
